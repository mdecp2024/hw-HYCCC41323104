1.

從 1 累加到 100

# 初始化總和為 0
total_sum = 0

# 使用 for 迴圈從 1 累加到 100
for i in range(1, 101):
total_sum += i

# 輸出結果
print("從 1 累加到 100 的總和是:", total_sum)

結果為: 5050 

Q:程式中這一段的意思?

# 使用 for 迴圈從 1 累加到 100
for i in range(1, 101):
total_sum += i

A:

把 <for i in range(1, 101): >分開介紹

  • for i 表示使用一個變數 i 來代表每一次循環中的當前數字。
  • in range(1, 101):
  1. range(1, 101) 是一個生成數字的範圍，表示從 1 開始，到 100 結束。這段語法會創建一個數字序列，包含從 1 到 100 的所有整數。
  2. 注意，range() 的第二個參數 101 是「不包含」的，所以它不會包括 101 這個數字。
  • 冒號則表示開始執行循環體內的程式碼。

<total_sum += i >的意思

  • 把變數 i 的值加到 total_sum 裡，然後把結果再存回 total_sum。total_sum 是 total_sum + i 的縮寫。

  • 這段程式碼的運作方式是這樣的：

    • 當 i = 1 時，total_sum += 1，total_sum 變成 1。
    • 當 i = 2 時，total_sum += 2，total_sum 變成 3。
    • 當 i = 3 時，total_sum += 3，total_sum 變成 6。
    • 當 i = 4 時，total_sum += 4，total_sum 變成 10。以此類推

addto

# 定義函式 addto
def addto(start, end):
# 初始化總和為 0
total_sum = 0

# 使用 for 迴圈從 start 累加到 end
for i in range(start, end + 1):
total_sum += i

# 返回累加的總和
return total_sum

# 呼叫函式並顯示結果
result = addto(1, 100)
print("從 1 累加到 100 的總和是:", result)

結果為: 5050

add_only_even 

# 定義函式 add_only_even
def add_only_even(start, end):
    # 初始化總和為 0，準備來累加偶數
    total_sum = 0
   
    # 用迴圈遍歷從 start 到 end 的每個數字
    for i in range(start, end + 1):
        # 檢查這個數字是否是偶數
        if i % 2 == 0:
            # 如果是偶數，就把它加到總和裡
            total_sum += i
   
    # 最後回傳累加的偶數總和
    return total_sum

# 呼叫函式 add_only_even，並顯示結果
result = add_only_even(1, 100)
print("從 1 累加到 100 的偶數總和是:", result)
  • 檢查每個數字是否是偶數：i % 2 == 0 是檢查 i 是否能被 2 整除，也就是檢查是否是偶數。
  • % 是 模數運算符，也叫做 取餘數。它用來計算一個數字除以另一個數字後，剩下的餘數。

結果為:從 1 累加到 100 的偶數總和是: 2550

1~100中避開有數字8的數字,例如 8，18，28，....。
add_avoid_8 

# 定義函式 add_avoid_8
def add_avoid_8(start, end):
 # 初始化總和為 0，準備來累加符合條件的數字
 total_sum = 0
 
 # 用迴圈遍歷從 start 到 end 的每個數字
 for i in range(start, end + 1):
 # 檢查這個數字是否包含數字 "8"
 if '8' not in str(i): # 把數字轉成字串，檢查是否包含 '8'
 # 如果沒有 '8'，就把這個數字加到總和裡
 total_sum += i
 
 # 最後回傳累加的總和
 return total_sum

# 呼叫函式 add_avoid_8，並顯示結果
result = add_avoid_8(1, 100)
print("從 1 累加到 100 且避開包含 '8' 的數字的總和是:", result)

結果為: 從 1 累加到 100 且避開包含 '8' 的數字的總和是: 3763

隨堂:

1~100不包含8和3

# 定義函式 add_avoid_8_and_3
def add_avoid_8_and_3(start, end):
# 初始化總和為 0，準備來累加符合條件的數字
total_sum = 0

# 用迴圈遍歷從 start 到 end 的每個數字
for i in range(start, end + 1):
# 檢查這個數字是否包含數字 "8" 或 "3"
if '8' not in str(i) and '3' not in str(i): # 把數字轉成字串，檢查是否包含 '8' 或 '3'
# 如果沒有 '8' 和 '3'，就把這個數字加到總和裡
total_sum += i

# 最後回傳累加的總和
return total_sum

# 呼叫函式 add_avoid_8_and_3，並顯示結果
result = add_avoid_8_and_3(1, 100)
print("從 1 累加到 100 且避開包含 '8' 或 '3' 的數字的總和是:", result)
